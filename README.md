# MachineLearning-DeepLearning

PDF Classification 

Demo - https://www.youtube.com/watch?v=pvP9nMXMyI8

This is an AI-powered document processing feature for the logistics sector. Inmplemented web UI using streamlit which has the capability to upload multiple documents.
These documents contains structured tables, semi-structured information, and unstructured text. The implemented feature is automatically process, classify these documents into invoice , bill of lading and other classes.
